title: Installation
description: Install Pomo on macOS, Windows, or Linux using your preferred package manager
---

Pomo is distributed as a single binary with no external dependencies. Choose your installation method below based on your operating system.

## Package managers

<CodeGroup>

```bash macOS (Homebrew)
brew install --cask bahaaio/pomo/pomo
```

```powershell Windows (Winget)
winget install Bahaaio.pomo
```

```bash Go install
go install github.com/Bahaaio/pomo@latest
```

</CodeGroup>

<Tip>
  The Homebrew installation uses the `--cask` flag to install the tap. The Go installation requires Go 1.21 or later.
</Tip>

## Pre-built binaries

Download pre-built binaries for your platform from the [GitHub releases page](https://github.com/Bahaaio/pomo/releases/latest).

Available platforms:
- **macOS** - Intel and Apple Silicon (M1/M2/M3)
- **Linux** - x86_64, ARM64, and ARM
- **Windows** - x86_64 and ARM64

<Steps>
  <Step title="Download the binary">
    Navigate to the [latest release](https://github.com/Bahaaio/pomo/releases/latest) and download the appropriate binary for your system.
    
    For example, on macOS:
    ```bash
    curl -LO https://github.com/Bahaaio/pomo/releases/latest/download/pomo-darwin-amd64
    ```
  </Step>
  
  <Step title="Make it executable">
    Add execute permissions to the binary:
    
    ```bash
    chmod +x pomo-darwin-amd64
    ```
  </Step>
  
  <Step title="Move to PATH">
    Move the binary to a directory in your PATH:
    
    ```bash
    sudo mv pomo-darwin-amd64 /usr/local/bin/pomo
    ```
  </Step>
  
  <Step title="Verify installation">
    Confirm Pomo is installed correctly:
    
    ```bash
    pomo --version
    ```
    
    Expected output:
    ```
    pomo version 1.1.1
    ```
  </Step>
</Steps>

## Build from source

For the latest development version or to contribute to Pomo, you can build from source.

### Prerequisites

- **Go 1.21 or later** - [Download Go](https://go.dev/dl/)
- **Git** - For cloning the repository

### Build steps

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/Bahaaio/pomo
    cd pomo
    ```
  </Step>
  
  <Step title="Build the binary">
    ```bash
    go build .
    ```
    
    This creates a `pomo` binary in the current directory.
  </Step>
  
  <Step title="Install (optional)">
    Move the binary to your PATH:
    
    ```bash
    # macOS/Linux
    sudo mv pomo /usr/local/bin/
    
    # Or add to your home bin directory
    mv pomo ~/bin/
    ```
  </Step>
  
  <Step title="Verify the build">
    ```bash
    pomo --version
    ```
  </Step>
</Steps>

<Note>
  Building from source gives you access to the latest features but may include unreleased changes. For production use, we recommend using a stable release.
</Note>

## System requirements

Pomo has minimal system requirements:

- **Operating system** - macOS 10.15+, Windows 10+, or any modern Linux distribution
- **Terminal** - Any terminal emulator with 256 color support
- **Font (optional)** - [Nerd Font](https://www.nerdfonts.com/) required for heatmap statistics display

### Terminal compatibility

Pomo works in all modern terminals:
- iTerm2, Terminal.app (macOS)
- Windows Terminal, PowerShell (Windows)
- GNOME Terminal, Konsole, Alacritty, Kitty (Linux)

## Uninstallation

To remove Pomo from your system:

<CodeGroup>

```bash macOS (Homebrew)
brew uninstall pomo
```

```powershell Windows (Winget)
winget uninstall Bahaaio.pomo
```

```bash Manual installation
# Remove the binary
sudo rm /usr/local/bin/pomo

# Remove configuration files (optional)
rm -rf ~/.config/pomo
```

</CodeGroup>
